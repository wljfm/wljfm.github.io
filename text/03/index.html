    个人网站的搭建

    2023年 1月 23日

    　

    准备工作

    1.搭建前请先注册一个github账号
    2.安装git、node.js、hexo、github上传插件
    　2(1).git安装 点击跳转
    　2(2).node.js安装 点击跳转
    　2(3).安装好node.js后在[cmd]中输入以下命令切换npm源

    1 |npm config set registry https://registry.npm.taobao.org

    　2(4).在cmd中输入以下命令安装hexo

    1 |npm install -g hexo-cli

    　2(5).在cmd中输入以下命令安装github上传插件

    1 |npm install hexo-deployer-git --save

    3.准备工作完成！

    网站搭建

    1.运行以下命令在当前目录(博客文件夹目录，而且必须为空目录)初始化一个使用hexo框架的项目

    1 |hexo init

    2.运行以下命令生成项目

    1 |hexo g

    3.运行以下命令启动本地服务器(按Ctrl+C停止运行)

    1 |hexo s

    4.右键项目目录打开Git bash here
    5.运行以下命令初始化github的邮箱和用户名

    1 |git config --global user.name "yourname"
    2 |git config --global user.email "youremail"

    6.运行以下命令生成ssh秘钥

    1 |ssh-keygen -t rsa -C "youremail"

    7.打开github，点击头像，在下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便取一个都可以，把你的ssh秘钥复制进去，点击create/创建
    8.输入以下命令确认是否连接成功(出现用户名则成功)

    1 |ssh -T git@github.com

    9.输入以下命令搭建网页(约两分钟即可访问)

    1 |hexo d

    10.如果上传失败请输入以下命令关闭git的上传证书认证，然后使用Watt Toolkit加速github后再次上传

    1 |git config --global http.sslVerify false
